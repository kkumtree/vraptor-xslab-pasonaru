## install(personal)

1. install microk8s

- Just Install microk8s and get usergroup
  - [Docs](https://microk8s.io/docs/getting-started)

```bash
sudo snap install microk8s --classic
sudo usermod -aG microk8s $USER
sudo chown -f -R $USER ~/.kube
su - $USER # re-enter the session
```
